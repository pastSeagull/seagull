    数据类型

    首先就是数据类型，基本数据类型：Boolean、Null、Undefined、Number、String，ES6加入的Symbol，还有ES2020的Bigint。还有复杂类型Object。
    Boolean、Number就不用说了，还有奇奇怪怪的Null和Undefined。对于基本类型可以使用typeof来检测，然后typeof null === 'object'，没错JS诞生以来就这样了。对于对象类型的检查可以使用Object.prototype.toString.call()。

    因为js中不同的对象在底层都表示为二进制，在 JavaScript 中二进制前三位都为 0 的话会被判 断为 object 类型， null 的二进制表示是全 0，自然前三位也是 0，所以执行 typeof 时会返回“ object ”。

    从内存来看看

    首先JS是一种弱类型的语言,就是你可以不用提前声明变量的类型，在程序运行中，它会被自动确定类型。

    基本数据类型是储存在栈里面的，而复杂类型是储存在堆的。

    let n1 = 123; // 不需要提前声明变量的类型
let n2 = n1; // 这里进行的是复制
let o1 = {}; // 这里是对堆地址的引用
let o2 = o1 // 这里是o1堆地址的引用
